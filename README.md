## ğŸš€ Getting Started

### Prerequisites

- Node.js 18.0 or higher
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd astro-cpm-calculator-v2/astrowind
   ```

2. **Install dependencies**
